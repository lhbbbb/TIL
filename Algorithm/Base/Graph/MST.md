# Minimum Spanning Tree(MST)

## Spanning Tree

* 최소 간선(n-1개)으로 그래프 내의 모든 정점을 연결하는 트리
* 하나의 그래프에서 스패닝 트리는 여러개가 존재할 수 있다

![spanning tree](assets\spanning_trees.jpg)

* 최소 신장 트리



## 프림

### 과정

1. 임의 정점을 하나 선택 시작
2. 선택한 정점과 인접하는 정점들 중 최소 비용의 간선이 존재하는 정점 선택
3. 모든 정점 선택될 때까지 1,2 반복
