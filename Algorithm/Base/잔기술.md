* 이중 배열 하나로 합치기

```python
a = [[1, 2], [3, 4], [5, 6]]
a
[[1, 2], [3, 4], [5, 6]]
sum(a, [])
[1, 2, 3, 4, 5, 6]
```

```python
[y for x in a for y in x]
```

* 딕셔너리 자료형 value를 기준으로 정렬하기

```python
lst = {2: 5, 3: 4, 1: 2, 6: 9}
sorted_lst = sorted(lst, key=lambda key: lst[key]) # 결과 key값만 보임
sorted_lst = sorted(lst.items(), key=lambda kv: kv[1]) # 결과 key, value 다 보임
```
