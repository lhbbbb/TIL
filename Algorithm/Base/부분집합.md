# 부분집합

* 완전탐색

## 부분집합의 합

```python
for i in range(1 << len(lst)): # 부분집합의 총 개수만큼 돌리기 2의 n승
	for j in range(len(lst)): # j는 bit 자리 수
        if i & (1 << j): # &비트 연산: 둘 다 1이면 True 그 외는 False
            print(lst[j], end=', ') # 해당하는 부분집합 출력 
    print()
```
