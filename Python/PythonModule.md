## Module

생산성과 재생산성을 높일 수 있음

### 표준 모듈

* 각기 목적에 맞게 설계되어 있고 다양한 함수, 클래스 등을 제공하며, 별도의 추가 설치 과정 없이 import 문으로 로딩해 사용함

=> 내장 모듈?

import 모듈명

import 모듈명 as 별칭

별칭으로 모듈 사용 가능

from 모듈명 import * 모듈 혹은 패키지가 가지고 있는 함수, 값 등의 모든 정보를 로딩해 사용 가능
